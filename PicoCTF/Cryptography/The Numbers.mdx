## Method

1. Download the provided image.
2. Identify the cipher: the numbers map to letters using **A1Z26** (A=1, B=2, ..., Z=26).
3. Decode the number sequence.
4. The decoded text contains the flag.

## Tools

- A1Z26 (letter-number) decoder: https://www.dcode.fr/letter-number-cipher
