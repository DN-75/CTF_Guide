## Method

1. Download the provided file.
2. Copy the encoded message from the file.
3. Decode it using **ROT13** (Caesar shift = 13).
4. The decoded text contains the flag.

## Tools

- Caesar / ROT13 decoder: https://www.dcode.fr/caesar-cipher

## Notes

ROT13 is a Caesar cipher with a shift of 13. Over the 26-letter alphabet (mod 26), applying ROT13 twice returns the original text.
